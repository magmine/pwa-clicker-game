# pwa-clicker-game

This is a Progressive Web App (PWA) game built with React and Lit.

## How to Run Locally

1. Install the dependencies:

```sh
npm install
```
2. Start the development server:

```sh
npm run dev
```

The application will be available at http://localhost:5173.

## Testing

1. To run the tests:

```sh
npm run test
```
